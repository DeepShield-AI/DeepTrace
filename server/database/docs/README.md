# Deployment

1. Set the `kibana_password` and `elastic_password` in the [configuration file](../../config/config.json)
2. Run: `python deploy.py`
3. Access: `http://ip:5601`  
   Username: `elastic`  
   Password: `elastic_password`

# Uninstallation
Run `python clear.py`
